1. Getting set up

First you need to create your ember app

  • ember new tutorial
  • cd tutorial
  • ember install liquid-fire
  • ember serve

2. Creating some great content

First generate an application template

ember generate template application

Then add some content to the application template

Initial application template
  • app/templates/application.hbs
  • Code
  • Diff 
Hi there!

Show some stuff?

{{input type='checkbox' checked=showStuff}}


{{#if showStuff}}
  <p>
    Here is some stuff
  </p>

  <ul>
    <li>A</li>
    <li>B</li>
    <li>C</li>
    <li>D</li>
  </ul>
{{/if}}

3. Adding animation

Simply change the if statement to a liquid-if statement, and you have animation!

Fancy animation
  • app/templates/application.hbs
  • Code
  • Diff 
Hi there!

Show some stuff?

{{input type='checkbox' checked=showStuff}}


{{#liquid-if showStuff use='to-left' reverse='to-right'}}
  <p>
    Here is some stuff
  </p>

  <ul>
    <li>A</li>
    <li>B</li>
    <li>C</li>
    <li>D</li>
  </ul>
{{/liquid-if}}
